# ホロライブ公式ショップ 検索＆一括ダウンロードツール

ホロライブプロダクション公式ショップ（shop.hololivepro.com）の  
「マイページ → ダウンロード商品」を爆速で探せて、快適にダウンロードできるユーザースクリプト

| スクリプト名                              | 対応環境                         | 主な機能                                                                 |
|------------------------------------------|----------------------------------|--------------------------------------------------------------------------|
| **PC（高機能フル版）**<br>`hololive-shop-dl-tool.user.js` | Tampermonkey     | ・購入履歴キャッシュ＋高速検索<br>・形式選択一括ダウンロード（wav/mp3/zip/pdf/画像）<br>・ダウンロード済みスキップ機能<br>・2秒間隔安定DL |
| **iPhone・iPad版（軽量検索特化）**<br>`hololive-shop-ios-search.user.js` | Userscriptsアプリ（無料）        | ・iOS Safariで完全動作<br>・全ページ自動取得＆ローカル保存<br>・タレント名＋フリーワード検索<br>・新着順／古い順ソート<br>・キーワード自由追加 |

どっちも**自分が買った商品だけ**を快適に整理・保存するためのファン製ツールです

## インストール方法

### PCの方（一括ダウンロードもしたい人）→ こっち推奨！
1. [Tampermonkey](https://www.tampermonkey.net/) をインストール  
2. このレポジトリのスクリプトのRAWを押してTampermonkeyに取り込むか[greasyfork](https://greasyfork.org/ja/scripts/558012-%E3%83%9B%E3%83%AD%E3%83%A9%E3%82%A4%E3%83%96%E5%85%AC%E5%BC%8F%E3%82%B7%E3%83%A7%E3%83%83%E3%83%97-dl%E5%95%86%E5%93%81%E6%A4%9C%E7%B4%A2-%E4%B8%80%E6%8B%ACdl%E3%83%84%E3%83%BC%E3%83%AB)からインストール！

→ インストール後、公式ショップのダウンロード一覧ページに行くと右下にボタン出現！

### iPhone・iPadの方（Safariで使いたい人）
1. App Storeで無料アプリ「[Userscripts](https://apps.apple.com/jp/app/userscripts/id1463298887)」をインストール  
2. Userscriptsを開いて「＋」→ このリポジトリの `hololive-shop-ios-search.user.js` を保存  
3. Userscriptsの設定で「常に許可」に設定  
4. Safariで公式ショップのダウンロード一覧ページを開くと、右下に丸いボタン出現！

## 機能詳細

### 共通機能
- 初回に全ページ自動読み込み → 以降は一瞬で検索可能（更新ボタンでいつでも最新化）
- ときのそら〜DEV_ISまで初期データ入り
- 「ASMR」「記念ボイス」「録り下ろし」などの便利タグ入り
- キーワードは自由に追加・削除可能

### PC版だけの超強力機能
- 詳細ページで「wav」「mp3」「zip」「pdf」「画像」をチェックボックスで選んで一括DL
- 過去にダウンロードしたファイルは自動で除外（重複防止）
- ファイル名が「商品名.wav」のように綺麗に保存
